## 📖 About the Project

The Last Minute Life Saver is an AI-powered productivity web application that helps users overcome procrastination and organize their work efficiently.

It assists users by:

- 🤖 Breaking complex tasks into smaller actionable steps
- 📅 Planning schedules intelligently
- 🔥 Prioritizing urgent work
- 🎯 Managing habits and productivity
- 🎤 Supporting voice interaction
- 💬 Providing AI coaching and suggestions

Built with React, TypeScript, Vite, and the Gemini API.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/harshm13/the-last-minute-life-saver.git
npm install
```
Create .env
GEMINI_API_KEY=YOUR_API_KEY

```bash
npm run dev
```
